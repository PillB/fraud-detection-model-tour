THE CHALLENGE

Why standard ML fails at fraud

Fraud incidence in financial transactions typically ranges from 0.01% to 2%. A trivial “all legitimate” classifier attains >99% accuracy while detecting zero fraud events.

Production-grade systems must detect known patterns (velocity bursts, mule accounts) alongside novel attacks (zero-day schemes), operate under extreme class imbalance, accommodate concept drift, and produce auditable, explainable scores for regulatory scrutiny.

METRIC OF CHOICE
PR-AUC
Focuses on the rare positive class. ROC-AUC is misleading on imbalanced data.
PRODUCTION PATTERN
Layered Cascades
Fast high-recall gate (rules + IF) → strong classifier → graph / generative → selective review.

Ensembles • SSL • Selective LLM

XGBoost
SUPERVISED BASELINE
Tier 1

Gradient-boosted trees constitute the production standard for labeled tabular fraud. Strong feature engineering—velocity, amount deviation, KYA aggregates—yields high PR-AUC on known patterns while delivering native feature importances and SHAP explanations.

Representative PR-AUC: 0.284
Velocity and KYA aggregates dominate importance rankings. Primary supervised layer; pairs naturally with anomaly scores.

VAE
GENERATIVE ANOMALY
Tier 3

VAE trained exclusively on legitimate transactions learns a probabilistic manifold of normal behavior. Per An & Cho (2015), reconstruction probability provides a calibrated anomaly score that surfaces novel and zero-day fraud where labels are absent. Latent representations also serve as high-value features for downstream supervised models.

Representative PR-AUC: 0.367 (VAE + IF hybrid)
An & Cho (2015) reconstruction probability. Latents integrate directly into hybrid pipelines.

MoE Hybrid
ADVANCED HYBRID
Tier 5–6

Mixture-of-Experts routes each input to specialized sub-models via a learned gate: RNN expert for behavioral sequences, Transformer/MLP for KYA interactions, AE expert for reconstruction anomalies. Vallarino (2025) reported 91.5% recall on high-fidelity synthetic credit-card fraud using this architecture.

Representative PR-AUC: 0.295 • 91.5% Recall (Vallarino 2025)
Dynamic routing adapts to fraud type. Leading example of adaptive hybrid systems.

LSTM Sequence
BEHAVIORAL
Tier 5

Recurrent networks model sequential dependencies in per-user transaction histories. Deviations in velocity, amount sequences, or merchant patterns surface even when single transactions appear innocuous. Essential for drift detection and behavioral fraud.

Representative PR-AUC: 0.261
Proxy leverages lagged features; full implementations use native per-entity sequences.

TabTransformer
TABULAR DL
Tier 4

Self-attention over categorical attributes (merchant category, KYA risk tiers, temporal bins) yields contextual embeddings. Detects higher-order interactions without manual cross-feature construction. Particularly powerful when KYA/KYE profiles supply rich categorical context.

Representative PR-AUC: 0.278
Educational embedding + classifier simulation; attention maps afford feature-level interpretability.

GraphSAGE
GRAPH / RELATIONAL
Tier 4

Inductive GNN that computes node embeddings through neighbor sampling and aggregation. Detects collusion rings, mule networks, and KYA-linked patterns invisible to tabular models alone. Inductive formulation supports previously unseen entities at inference.

Representative PR-AUC: 0.319 (proxy)
NetworkX mean-aggregation proxy on synthetic graph. Production implementations use PyG or DGL.

Documented Pipeline

scripts/full_pipeline.py
View Source
Fully commented • Step-by-step rationale • Reproducible
1. Load & Clean
Generate synthetic transactional + entity + graph data. Clean missing values. Profile distributions and injected fraud archetypes (velocity, amount deviation, relational). Reproducible with fixed seeds.
2. Feature Engineering
Core signals: velocity (1h/24h counts), amount statistics vs. history, temporal (hour, night), KYA risk proxies. StandardScaler + OneHot. Feature engineering routinely outperforms model swaps (Bahnsen et al.).
3. Modeling
Parallel training across families: supervised (XGBoost), tabular proxies, Isolation Forest (anomaly gate), ensemble meta-learner. Covers known patterns, novel deviations, and relational signals.
4. Ensemble + Output
Late score fusion (supervised + anomaly). Report PR-AUC for ensemble and hybrid. Rank highest-risk transactions with multi-view explanations (importance, reconstruction, attention). Foundation for production cascades.
Run: python scripts/full_pipeline.py. The script emits documented rationales per step, Ensemble PR-AUC, Hybrid PR-AUC (typically 0.29–0.35 range on toy data), and ranked top transactions with explanations. Core Insights from Research & Experiments
  • • Feature engineering—velocity windows, behavioral aggregates, KYA/KYE proxies—delivers larger marginal gains than model family changes (Bahnsen et al.).
  • • Hybrids dominate production: generative components (VAE) surface novel patterns; supervised (XGBoost) address known archetypes; graph layers close relational blind spots.
  • • Layered cascades (fast classical gate → specialized DL/graph → selective review) are the established pattern across 2024–2026 deployments.
  • • PR-AUC and Recall@K are the operationally relevant metrics; accuracy and ROC-AUC are actively misleading under extreme imbalance.
Decision Framework: Model Selection
Foundation layer: XGBoost + Isolation Forest on velocity + KYA features.
Novel / zero-day coverage: VAE reconstruction probability or IF hybrids.
Rich categorical context: TabTransformer contextual embeddings.
Behavioral drift & sequences: LSTM / sequence models or proxies.
KYA/KYE relational fraud: GraphSAGE (or TGN in production).
Mixed or adaptive requirements: MoE hybrids or full cascaded ensembles.
